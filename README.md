# docker mysql server

Запустите Mysql сервер

`docker-compose up -d`

Сервер будет доступен по порту `3306`

Adminer будет доступен `8080`

Файл конфига my.cnf в папке conf.d

Если выходите при запуске контейнера, воостановить БД из дампа,

загрузите перез запуском контейнера в папку `data` дамп бд (sql или sql.tar.gz)
